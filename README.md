# sample-spring-batch
Spring-Batch + Spring-Bootを利用したサンプルバッチです。

## 動作確認環境
| Platform | Version |
| -------- | ------- |
| Java | 8 |
| DB2 | 11.1 |

## 利用技術
| FW/OSS | Version |
| ------ | ------- |
| Spring Boot | 2.1.4.RELEASE |
| Spring Batch | 4.1.2.RELEASE |
| mybatis-spring | 2.0.1 |
| mybatis | 3.5.1 |

## 機能
- Rest Service呼び出しジョブ(CallRestService)
- DB to File(DbToFile)
- File to DB(FileToDb)
